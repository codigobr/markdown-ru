Bear

Bear - это простое приложение для создания заметок для устройств macOS и iOS.

Вступление

Bear - очень простое в использовании приложение, оно простое, гибкое и быстрое, а также отвечает основным требованиям для редактирования документов с помощью Markdown.

Bear позволяет использовать tags, синхронизацию файлов и преобразование документов.

Но пусть вас не обманывает его простота: с Bear вы можете создавать как простые документы, так и более сложные эссе.

Посетите официальный веб-сайт Bear , нажав здесь.

Использование Markdown в Bear

Используйте меню настроек, чтобы включить поддержку Markdown в приложении.

В редакторе реального времени Bear отображает результат вашей разметки Markdown по мере того, как вы набираете документ.

Если вы только начинаете работать с Markdown, просмотр изменений в реальном времени поможет вам привыкнуть к языку.

Включение поддержки Markdown

Сначала откройте окно настроек Preferences, перейдите на вкладку General и выберите вариант Markdown compatibility mode.

Вы должны сделать это, включив совместимость с Markdown для каждого экземпляра Bear на всех ваших устройствах.
